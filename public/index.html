urlshot uses phantomjs and nodejs to render images of webpages.

Example: /?url=http://nytimes.com/

Loaded example: /?url=http://reddit.com/&viewport=500x900&scrollto=200&format=jpg

Params: 
       url: The webpage to render an image of.   Required.
  viewport: The size of the browser "window".    default: 1024x768
  scrollto: Number of pixels to scroll down.     default: 0
  fullpage: Render the whole webpage.            default: false
    format: Type of image to render. png or jpg. default: png

It is recommended that you install and run your own instance of urlshot; linking to images on the free heroku-hosted demo app is a recipe for 500s.
